# lion-ui

基于 vue+element_ui 开发的 RBAC 管理系统

[文档教程](http://book.levy.net.cn/doc/frontend/uiframe/env.html)

# 使用

1. 克隆代码
2. 安装相关依赖  npm i
3. 启动测试环境 npm run serve
**tips:master分支代码使用的 fastmock 的数据，拉下来可直接使用的，可切换配置文件中的地址。mock数据 格式在教程中都有**
